# Tactile logistics interface

Handoff looks like a parcel deal sheet and inspection station, not a crypto dashboard or game. The paper ground is `#F2E9D8`, carbon ink `#18201C`, dispatch blue `#1D4ED8`, safety orange `#F05A28`, tape yellow `#F4C95D`, verified green `#207A52`, and muted ink `#66706A`.

Headings use self-hosted Bricolage Grotesque, interface copy uses Manrope, and identifiers/amounts use Azeret Mono. Perforations, barcodes, dashed receipt rules, tape status, scan frames, and terminal stamps create the physical metaphor. Odd One’s dark palette, game-show language, orbit layouts, cards, motion, typography, and colors are intentionally absent.

Mobile provides a compact sticky app header and single-column inspection route; desktop pairs the private deal sheet with a sticky action station. Controls have at least 44px targets, visible focus, semantic headings, live status, plain-language financial warnings, and reduced-motion overrides. Color is never the only state indicator.

The landing interaction is explicitly marked `Local preview · not live`. Missing deployment configuration never creates fake contract data.
